## Income Statement 利润表

营业利润 = 营业收入 - 营业成本 - 销售费用 - ...
利润总和 = 营业利润 + 营业外收入 - 营业外支出
净利润 = 利润总和 - 所得税费用
(综合收益总额 = 净利润 + 其他综合收益的税后净额)
